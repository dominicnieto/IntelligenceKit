The AI SDK comes with several providers that you can use to interact with different language models.

## Swift provider modules

The Swift AI SDK ships with **11 provider modules** (one SwiftPM product per provider).

| Provider | SwiftPM product | Docs |
| --- | --- | --- |
| OpenAI | `OpenAIProvider` | [OpenAI](/providers/openai) |
| Anthropic | `AnthropicProvider` | [Anthropic](/providers/anthropic) |
| Google Generative AI | `GoogleProvider` | [Google Generative AI](/providers/google-generative-ai) |
| Google Vertex AI | `GoogleVertexProvider` | — |
| Vercel | `VercelProvider` | [Vercel](/providers/vercel) |
| Mistral | `MistralProvider` | [Mistral](/providers/mistral) |
| xAI | `XAIProvider` | — |
| Perplexity | `PerplexityProvider` | [Perplexity](/providers/perplexity) |
| Gateway | `GatewayProvider` | — |
| OpenAI Compatible | `OpenAICompatibleProvider` | — |
| ElevenLabs | `ElevenLabsProvider` | — |
